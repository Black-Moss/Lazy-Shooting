# Lazy Shooting

[GitHub](https://github.com/Black-Moss/Lazy-Shooting) / [Nexus Mods](https://www.nexusmods.com/scavprototype/mods/14)

Hold your left mouse button to shoot through the layers!

__Requires [Moss Lib](https://github.com/Black-Moss/Moss-Lib) as a dependency!__

Change configuration properties in the config file or use the `lazyshooting` command!

_Using `lazyshooting` to change configurations will sync to the config file._

## Features

* Ammunition UI (`ammunitionui`): Display your ammunition in real time!
* Auto Srack (`autosrack`): If true, guns will automatically rack and stay racked when ammo is available.
* Indestructible Gun (`indestructiblegun`): If true, guns will not be destroyed.
* Infinite Ammunition (`infiniteammunition`): ∞ INFINITE AMMUNITION ∞
* Never Jam (`neverjam`): If true, guns will never jam.
* Recoiless (need restart game) (`recoiless`): If true, guns will not have recoil.
